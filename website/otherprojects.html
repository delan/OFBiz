Open Source Projects

The Open For Business Project also uses many other open source projects to take advantage of some of the great work that is being done in other parts of the open source community. Some of these are used directly and would be somewhat difficult to replace while others are used through standard Sun or other interface specifications such as the Servlet API or JAXP. The open source projects used in OFBiz include:

  • Web Services Publisher (WSP)
  • BeanShell
  • DataVision
  • JasperReports
  • WEKA
  • Apache
    • XML
      • Xerces
      • Xalan
      • Axis
    • Jakarta
      • Ant
      • Tomcat 4
      • Log4J
      • ORO
  • Exolab
    • Castor
    • Tyrex
  • JBoss
  • PostgreSQL
  • MySQL
  • Hypersonic SQL
